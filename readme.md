## Given tasks:

- System for items organization
- Each item must have:
    - `Title`
    - `Count`
    - `Price`
    - `Category`
    - `Description is optional`
    
- Functions to implement:
- `Items creation`
- `Items review`
- `Categories creation`
- `Categories review`
- `Items which belongs to specific category review`
- System must have one public endpoint which returns items information as JSON
    - `/api/items/{id}`
